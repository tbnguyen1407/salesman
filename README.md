## Development

### Requirements

- .NET Framework 4.8.1
- Build Tools for Visual Studio 2019+ (for building)

### Build

```shell
./compile.bat
```

### Publish

```shell
./publish.bat
```

## Running

### Requirements

- .NET Framework 4.8.1
- SumatraPDF (for printing support, put the executable in `native` folder)
